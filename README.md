## 📌 Project Overview

This project analyzes retail sales data using Tableau to generate meaningful business insights. 
The dashboard provides an interactive view of sales, profit, orders, and category-wise performance.

## 🎯 Objectives

- Analyze overall sales and profit performance
- Identify Top 5 products by sales
- Compare category-wise performance
- Visualize order distribution using histogram
- Implement interactive filters and parameters

- ## 🛠 Tools Used

- Tableau Desktop
- Excel (Data Source)

- ## 🚀 Key Features

- Interactive Dashboard
- Filters and Parameters
- Grouping of Products
- Top N Analysis
- Histogram Visualization
- KPI Cards for Sales, Profit, Orders

- ## 📈 Key Insights

- Identified highest revenue generating products
- Observed category-wise profit variations
- Analyzed order frequency distribution
- Compared performance across different product categories

- ## ✅ Conclusion

This project demonstrates my ability to analyze business data, create interactive dashboards, and generate actionable insights using Tableau.
